# Real-Time Data Pipeline

## 📌 Project Overview
This project simulates a real-time data pipeline using Java.

## ⚙️ Features
- Producer-Consumer architecture
- Multithreading
- Real-time data processing
- MySQL integration using JDBC
- Revenue analytics

## 🛠 Technologies
- Java
- MySQL
- JDBC
- Multithreading

## ▶️ How to Run

Compile:
javac -cp ".;mysql-connector-j-9.6.0.jar" OrderPipeline.java

Run:
java -cp ".;mysql-connector-j-9.6.0.jar" OrderPipeline

## 📊 Output
- Real-time order processing
- Data stored in MySQL
- Revenue calculation
